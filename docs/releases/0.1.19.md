# Release 0.1.19 — polish the quiet garden and lighten the load

Turn the June UI review into a shippable polish pass: remove misleading or duplicated journey chrome, make destructive actions explicit, give archive/settings/home the same craft as the garden, and convert runtime sprites to WebP without losing the companion's alpha quality.

## Highlights

- Fixes the seed title sentence so user text is no longer grafted into "I am noticing the story that..."; new seeds now read `Noticing the story: “…”`.
- Adds inline confirmation for deleting seeds and for resetting a lens profile while a journey draft exists.
- Restores the top-bar theme control to three states: light, dark, and match system.
- Hides duplicate garden status/progress chrome while the lens panel is open, while keeping the rail available when a journey is dismissed mid-step.
- Gives the archive a real empty state, growth-stage art on seed cards, and friendly planted dates such as "today", "yesterday", and "4 days ago".
- Refines the settings and home screens with grouped settings sections, clearer privacy copy, adaptive home actions, and display typography through Fraunces.
- Converts runtime companion frames, garden props, and lens props from PNG to WebP, keeping companion frames lossless while using audited lossy WebP for props and lenses.

## Lore

Constraint: Ship the review polish without changing the lens journey's underlying model or relaxing the asset audits that protect alpha edges and crop margins.
Rejected: Use browser `confirm()` for destructive actions | Inline confirmation keeps the flow styled, testable, accessible, and explicit about what will be discarded.
Rejected: Convert companion frames with lossy WebP | The payload savings were tempting, but lossy and near-lossless modes shifted semi-transparent pixels enough to fail the pet matte audit.
Rejected: Leave generated source sheets under `src/assets` | They are important provenance, but they should not sit beside runtime-imported files or confuse bundle-facing asset scans.
Rejected: Keep only light/dark in the top bar | Settings already support following the system theme; the global control should be able to return to that state without opening another screen.
Confidence: high
Scope-risk: medium
Directive: Let the garden feel quieter, make every dangerous action ask plainly, and make the shipped assets lighter without thinning their edges.
Tested: `pnpm lint`; `pnpm typecheck`; `pnpm test` (92 tests, 12 files); `pnpm build`; `pnpm format:check`; `pnpm lens:audit`; `pnpm pet:audit`; `pnpm garden:audit:dark`; `pnpm exec playwright test --project=chromium` (29 passed).
Not-tested: GitHub Actions production deploy and release publishing have not run for this tag yet; full cross-browser Playwright projects and a physical-device seven-lens journey remain on the beta checklist.
