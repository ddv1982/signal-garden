# Release 0.1.30 — open the mobile gate

Let the same garden path run on mobile instead of keeping most of it behind desktop-only skips: the full Playwright loop now runs for mobile Chrome, the old smoke-only gate is retired, and debug-only skipped checks are pruned.

## Highlights

- Removes mobile project skips from the normal garden journey, planting, Settings, pet, and storage E2E coverage.
- Replaces the `@mobile-smoke` lane with a full `mobile-chrome` Playwright script and CI job.
- Deletes debug-harness tests that were skipped or only guarded local pose-review behavior.
- Makes E2E helpers work across desktop and mobile by using touch taps on mobile canvases, viewport-aware planting coordinates, and deterministic lens form submission.
- Keeps the release suite free of expected skips: desktop Chromium and mobile Chrome now both run the same 31-test garden loop.

## Lore

Constraint: The mobile gate needed to walk the real path, but the change could not loosen product assertions or keep skipped tests as ornamental proof.
Rejected: Keep the smoke test as a separate mobile talisman | Once the full mobile suite passes, a one-test smoke lane only adds a desktop skip and hides less than it claims.
Rejected: Preserve debug pose checks with more conditions | A permanently skipped local visual harness is not release confidence; user-facing pet and reduced-motion behavior remain covered.
Rejected: Reintroduce broad mobile skips for canvas-sensitive flows | The better fix was to make the helpers speak mobile input and mobile plot coordinates directly.
Confidence: high
Scope-risk: low
Directive: Open the mobile gate and let every living path prove itself twice.
Tested: `pnpm e2e:compiled --project=chromium` (31 passed, 0 skipped); `pnpm e2e:mobile:compiled` (31 passed, 0 skipped); `pnpm format:check`; `pnpm lint`; `pnpm typecheck`; `pnpm test` (117 tests, 19 files); `pnpm build`.
Not-tested: GitHub Actions production deploy and release publishing have not run for this tag yet; physical iOS Safari and macOS Safari were not available in this release pass.
