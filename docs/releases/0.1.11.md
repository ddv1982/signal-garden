# Release 0.1.11 — restore the sky edge

Correct the 0.1.10 garden-edge fix after the render-time crop removed the magenta artifact but exposed Phaser's plain scene background as a beige strip above the sky.

## Highlights

- Restores the full garden background draw so the painting keeps its original alignment.
- Patches only the contaminated top strip with clean sky pixels from the same background image.
- Keeps plot placement, lens objects, signal placement, pet positioning, and pointer targets on the unchanged full design frame.
- Verifies the top edge now starts as painted sky, with no magenta line and no beige bar.
