# Release 0.1.7 — smooth the body ripple

Repair the body-ripple lens prop after visual QA found a clipped edge and colored matte, and harden the lens asset cleanup path so future generated props stay stable.

## Highlights

- Re-cuts the body-ripple/orb prop with the full pool visible and transparent padding around the artwork.
- Removes leftover chroma-key matte pixels from lens props without treating real green foliage detail as background.
- Adds stable alpha feathering and transparent-edge color bleed so canvas filtering does not show harsh or tinted edges.
- Makes the lens cleanup script byte-for-byte idempotent after the first clean pass.
- Verifies the active body lens in the Phaser garden on desktop and mobile viewports.

## Lore

Constraint: Generated lens props should look handmade in the garden without carrying source-sheet matte, hard crop edges, or cleanup drift.
Rejected: Hide the body-ripple cutoff with Phaser scale or CSS | The crop and matte were baked into the runtime PNG, so the asset and pipeline needed repair.
Rejected: Treat green-tinted edge concerns as green-screen removal | The source sheet uses magenta keying, and green thresholds risk deleting real foliage details.
Confidence: high
