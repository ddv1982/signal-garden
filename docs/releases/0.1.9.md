# Release 0.1.9 — clear the garden edges

Clean the visible matte and crop artifacts around the companion and lens props after visual QA found a green cat edge, a bright cloud halo, and lingering pool cutout tint in the Phaser garden.

## Highlights

- Warms and defringes every companion frame so the pet no longer carries green/olive edge matte into the garden.
- Re-cuts the image-cloud and observer-pool lens props from the chroma-key source sheet with full transparent padding.
- Softens cloud halo pixels and removes lower-edge pink matte from the observer pool while keeping the watercolor silhouette intact.
- Keeps the body-ripple and other lens props on the hardened cleanup path so transparent edges do not sample stale chroma-key color.
- Adds pet-frame edge-matte QA and extends lens crop QA so future prop exports catch these failures automatically.
- Verifies the corrected props in the live Phaser scene on desktop and mobile lens states.

## Lore

Constraint: Handmade garden assets should keep their soft watercolor edge without carrying source-sheet matte, stale transparent RGB, or hard crop scars into play.
Rejected: Hide the green cat edge with scene tinting or background changes | The fringe was baked into the companion PNGs and had to be cleaned at the asset boundary.
Rejected: Remove the cloud halo by cutting deeper into the cloud | A destructive trim punched holes in the soft cloud silhouette, so the fix keeps the cloud form and only softens halo pixels at transparent neighbors.
Rejected: Shrink the observer-pool PNG to avoid overlap | The art needed the full recovered pool; only the scene display size changed slightly after the asset ratio became accurate.
