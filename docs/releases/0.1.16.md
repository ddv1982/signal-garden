# Release 0.1.16 — ground the moonlit focus

Polish the dark garden's runtime lighting so the companion and active lens objects feel grounded in the nighttime scene instead of surrounded by visible selection halos.

## Highlights

- Removes the broad dark-mode oval behind the companion while preserving a tighter ground shadow.
- Replaces the active dark lens ring and glint with a low footprint glow plus small motes.
- Keeps light mode, lens hit targets, texture keys, generated assets, and saved-state contracts unchanged.
- Preserves reduced-motion behavior by keeping the new dark focus effect static when motion is reduced.

## What changed

- Release metadata:
  - `package.json`
  - `docs/releases/README.md`
  - `docs/releases/0.1.16.md`
  - Bumps the package release line to `0.1.16` and adds this note to the release index.
- Garden runtime:
  - `src/game/GardenGame.ts`
  - Removes the dark companion body halo that created a visible oval around the cat.
  - Replaces dark active-lens rim settings with grounded focus and mote lighting settings.
  - Draws the active focus below the lens prop and draws only small motes above it.

## Lore

Constraint: Dark-mode polish should remove visible selection halos without regenerating art, moving assets, changing hit targets, or changing the light-mode experience.
Rejected: Keep the cat body aura | It helped separate the companion from the night background, but it read as a circle around the cat.
Rejected: Keep the active stone rim | It made the selected word stones clear, but it looked like a UI outline drawn over the artwork.
Rejected: Regenerate the companion and lens assets | The issue was runtime Phaser lighting, and new assets would add anchor and alpha risk for a focused polish release.
Rejected: Remove all active lens feedback | The active lens still needs to be discoverable and tap-safe on desktop and mobile.
Confidence: high
Scope-risk: low
Directive: Let dark-mode focus come from the ground and small night lights, not from visible rings around the art.
Tested: `pnpm test -- --run`; `pnpm exec tsc --noEmit`; `pnpm run build`; targeted Playwright desktop/mobile checks for active lens interaction and system dark theme; in-app Browser visual checks for desktop and 390px mobile dark garden rendering.
Not-tested: GitHub Actions production deploy and release publishing have not run for this tag yet.
