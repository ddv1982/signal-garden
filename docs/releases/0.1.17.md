# Release 0.1.17 — balance the dusk garden

Restore the dark garden's composition after the night pass drifted too far from the daytime stage, keeping the warmer dusk mood while bringing the pond, path, and prop spaces back into balance.

## Highlights

- Replaces the runtime dark backdrop with a dusk/night image pass that keeps the current atmosphere while reducing the oversized pond.
- Keeps the garden composition closer to the daytime backdrop so desktop and mobile crops feel familiar across themes.
- Retunes companion, lens, and garden prop dark assets toward the warmer dusk palette while preserving dimensions, alpha bounds, and Phaser anchors.
- Keeps the dark lens chip menu wrapping cleanly on narrow layouts without a visible horizontal scrollbar.
- Records generated dark backdrop sources and cleanup notes for future art passes.

## Lore

Constraint: The dark garden should keep the current dusk/night feeling, but the pond and stage layout should read closer to the daytime composition.
Rejected: Return to the deterministic relit daytime image | It preserved geometry, but lost the richer nighttime atmosphere the garden now needs.
Rejected: Keep the v2 dusk backdrop unchanged | It was more beautiful than the first dark pass, but the pond became too large and changed the stage balance.
Rejected: Regenerate foreground assets from scratch | The existing transparent assets already had stable anchors, so a profiled dusk retune was safer than new cutouts.
Rejected: Force chip tabs into a horizontal scroller | It avoided wrapping, but recreated the visible scrollbar problem on Linux Chrome.
